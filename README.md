# My-Vim-Setup
This is my Vim setup.

Step to use my vim configuration:

For this configuration, you must have installed the latest version of Node.js ([My installation script](https://github.com/SaruMan64/Install-Node.js) or [official website](https://nodejs.org/en/download/)) and npm.

```console
$ git clone https://github.com/SaruMan64/My-Vim-Setup.git
$ cd My-Vim-Setup/
$ chmod 755 vimConfig.sh
$ ./vimConfig.sh
```
***Warning***: Don't press enter and/or other keys.
